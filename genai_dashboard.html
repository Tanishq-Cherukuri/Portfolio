Here's the DAX measure for Treatment Cost YoY Variance %:
DAX — Generated by CopilotCopy ⧉
Treatment Cost YoY % =
VAR CurrentCost =
  CALCULATE(SUM(Facts[TreatmentCost]))
VAR PriorCost =
  CALCULATE(SUM(Facts[TreatmentCost]),
  SAMEPERIODLASTYEAR(DimDate[Date]))
RETURN
  DIVIDE(CurrentCost - PriorCost, PriorCost, 0)
